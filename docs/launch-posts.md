# Linoa ローンチ告知ドラフト

## X（Twitter）投稿案

### パターンA: 課題提起型
```
飲食店を経営しているみなさん、

「日報つけたいけど忙しくて続かない」
「売上の傾向、なんとなくしかわからない」

そんな悩みを解決するために、LINEで使えるAI秘書「Linoa」を作りました。

LINEで話しかけるだけで日報完了→売上分析→改善提案まで。

現在無料テスト中です。
https://li-noa.jp
```

### パターンB: 開発ストーリー型
```
飲食店オーナー向けのAI秘書「Linoa」をリリースしました。

LINEで日報を送るだけで:
・売上/客数をグラフで可視化
・AIが経営のヒントを提案

「アプリを入れたくない」「PCは苦手」という声から、LINEだけで完結する設計にしました。

無料テスト中なので、飲食店経営者の方はぜひ。
https://li-noa.jp
```

### パターンC: シンプル型
```
飲食店の「店舗専属AI秘書」Linoaを公開しました。

LINEで質問に答えるだけで日報完了。
売上の傾向分析も自動。

現在無料でテスト利用できます。
https://li-noa.jp
```
